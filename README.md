# aquacare
Manage Aquariums & Fishes with <b>Aquacare</b>
<h4>Built using:</h4>
<ul>
  <li>Android Java</li>
  <li>SQLite</li>
</ul>
<h4>Features</h4>
<ul>
  <li>Choose photo for aquarium from <b>Camera</b> or <b>Gallery</b>.</li>
  <li>Create, Read, Update & Delete Aquarium.</li>
  <li>Create, Read, Update & Delete Fish.</li>
  <li>Search Fish</li>
</ul>
